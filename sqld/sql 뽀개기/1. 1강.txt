1. 정의
  1) 데이터
    - 실제 세상에 넓게 존재하는 사실적인 자료

  2) 정보
    - 이러한 데이터가 의미 있는 패턴으로 정리되면 정보가 됨
    - Information = processed data

  3) 지식
    - 정보가 일반화된 형태로 정리된 것
    - 의사결정이나 창출에 이용해 부가가치 발생

  4) 지혜
    - 지식을 얻고 이해하여 응용해 발전해 나가는 정신적 능력

2. 존재적 특징
  1) 정성적 데이터
    - 데이터 하나하나가 함축된 의미를 갖고 있음
    - 비정형 데이터

  2) 정량적 데이터
    - 여러 속성(이름, 나이, ...)이 모여 하나의 객체를 형성
    - 정형 데이터

3. 형태적 특징
  1) 정형 데이터  
    - 데이터의 구조(스키마)가 미리 정의된 데이터
        - 스키마 : 저장하는 데이터의 구조
    - DBMS에 스키마를 관리하는 DB(Dictionary)가 
      별도로 존재하여 DB 저장소와 구분됨
    - Type, Length가 미리 정의됨      

  2) 반정형 데이터
    - 데이터 내부에 스키마에 해당되는 메타 데이터를 갖고 있음
    - 일반적으로 파일 형태로 저장됨
    - JSON, XML, HTML 등등

  3) 비정형 데이터
    - 하나의 데이터가 수집 데이터로 객체화되어 있는 데이터
    - 텍스트, 이미지, 동영상, 등등
